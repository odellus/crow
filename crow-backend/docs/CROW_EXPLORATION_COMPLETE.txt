QUICK START

To run the server:
  $ cd crow/packages/api
  $ cargo run --bin crow-serve --features server

To create a session:
  $ curl -X POST http://127.0.0.1:7070/session \
    -H "Content-Type: application/json" \
    -d '{"title": "My Session"}'

To send a message:
  $ curl -X POST http://127.0.0.1:7070/session/ses-{id}/message \
    -H "Content-Type: application/json" \
    -d '{"agent": "default", "parts": [{"type": "text", "text": "Hello"}]}'
